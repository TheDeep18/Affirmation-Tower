A playful, game-like web app for reframing negative thoughts into positive affirmations. Built with React, TypeScript, and Framer Motion.

## Features

- **Swipe Gameplay**: Reject negative thoughts (swipe left) and accept affirmations (swipe right).
- **Tower Building**: Accepted affirmations become blocks to build a physical tower.
- **Physics**: Balance your tower and avoid collapse with wobble mechanics.
- **Progression**: Daily streaks, badges, and evolving backgrounds.
- **Accessibility**: Full keyboard support and screen reader friendly.
- **Privacy**: All data is stored locally on your device.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion (Animations)
- Zustand (State Management)
- Vitest (Testing)

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start development server**
   ```bash
   npm run dev
   ```

3. **Run tests**
   ```bash
   npm test
   ```

## How to Play

1. **Daily Mode**: A curated set of 12 cards to start your day.
2. **Free Play**: Unlimited mode to keep building.
3. **Controls**:
   - **Swipe**: Touch or Mouse drag.
   - **Keyboard**: Left Arrow (Reject), Right Arrow (Accept), Space (Drop Block).

## Accessibility

- High contrast mode available in settings.
- Reduced motion preference respected.
- Focus indicators for keyboard navigation.
